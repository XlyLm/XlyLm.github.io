JS数组API

发表于2022-06-02|更新于2022-06-02|JavaScript
|字数总计:2k|阅读时长:8分钟|阅读量:

一、ES5数组API

1、constructor

返回数组的构造函数

其返回值是对函数的引用，而不是函数的名称：

对于 JavaScript 数组，constructor 属性返回：
function Array() { [native code] }

对于 JavaScript 对象，constructor 属性返回：
function Object() { [native code] }

2、length

设置或返回数组中元素的数量。

3、prototype

允许您向数组添加属性和方法。

4、concat

连接两个或多个数组，并返回已连接数组的副本。

方法不会更改现有数组，而是返回一个新数组，其中包含已连接数组的值 .

语法： array1.concat(array2 [, array3, …, arrayX])

5、every

检查数组中的每个元素是否通过测试。

方法对数组中存在的每个元素执行一次函数 。

所有元素通过测试，enery()返回true，否则返回false。

注: 不对没有值的数组元素执行函数。

语法： array.every(function(currentValue, index, arr) [, thisValue])

6、filter

使用数组中通过测试的每个元素创建新数组。

注： 不会对没有值的数组元素执行该函数。

语法： array.filter(function(currentValue, index, arr) [, thisValue])

7、forEach

按顺序为每个数组元素调用一次函数。

注： 对于没有值的数组元素，不执行forEach() 方法。

语法： array.forEach(function(currentValue, index, arr) [, thisValue])

8、indexOf

在数组中搜索元素并返回其位置。

搜索将从指定位置开始，如果未指定开始位置，则从头开始，并在数组末尾结束搜索。

如果未找到该项目，则 indexOf() 返回 -1。

如果该项目出现多次，则 indexOf() 方法返回第一次出现的位置。

注： 第一项的位置为 0，第二项的位置为 1，依此类推。

语法： array.indexOf(item [, start])

9、isArray

检查对象是否为数组。

如果对象是数组，则此函数返回 true，否则返回 false。

语法： Array.isArray(obj)

10、join

将数组的所有元素连接成一个字符串。

元素将由指定的分隔符分隔。默认分隔符是逗号 (,)。

语法： array.join(separator)

11、map

使用为每个数组元素调用函数的结果创建新数组。

注： 对没有值的数组元素不执行函数。

语法： array.map(function(currentValue, index, arr) [, thisValue])

12、pop

删除数组的最后一个元素，并返回该元素。

注： 方法会改变数组的长度。

语法： arr.pop()

13、push

将新元素添加到数组的末尾，并返回新的长度。

注： 方法会改变数组的长度。

语法： array.push(item1 [, item2, …, itemX])

14、reduce

将数组的值减为单个值（从左到右）。

函数的返回值存储在累加器中（结果/总计）。

注： 对没有值的数组元素，不执行 reduce() 方法。

语法： array.reduce(function(total, currentValue, currentIndex, arr) [, initialValue])

15、reverse

反转数组中元素的顺序。

语法： array.reverse()

16、shift

删除数组的第一个元素，并返回该元素。

注： 方法会改变数组的长度。

注： 方法的返回值是被移除的项目。

语法： array.shift()

17、slice

选择数组的一部分，并返回新数组。

方法选择从给定的 start 参数开始的元素，并在给定的 end 参数处结束，但不包括。

语法： array.slice(start, end)

18、some

检查数组中的任何元素是否通过测试。

如果找到函数返回真值的数组元素，some() 返回真 ，否则返回false。

注： 对没有值的数组元素不执行函数。

语法： array.some(function(currentValue, index, arr) [, thisValue])

19、sort

对数组的元素进行排序。

排序顺序可以是按字母或数字，也可以是升序（向上）或降序（向下）。

默认情况下，sort() 方法将按字母和升序将值作为字符串进行排序。

语法： array.sort(compareFunction)

20、splice

从数组中添加/删除元素。

语法： array.splice(index, howmany [, item1, ….., itemX])

21、toString

将数组转换为字符串，并返回结果， 以逗号分隔 。

语法： array.toString()

22、unshift

将新元素添加到数组的开头，并返回新的长度。

注： 方法会改变数组的长度。

语法： array.unshift(item1 [, item2, …, itemX])

二、ES6数组API

1、entries

返回键/值对数组迭代对象。

语法：array.entries()

2、fill

用静态值填充所有数组元素 。

您可以指定开始和结束填充的位置。如果未指定，则将填充所有元素。

语法： array.fill(value, start, end)

3、find

返回数组中第一个通过测试的元素的值。

如果找到函数返回 true 值的数组元素，则 find() 返回该数组元素的值 ，否则返回undefind。

注： 不对空数组执行该函数。

语法： array.find(function(currentValue, index, arr) [, thisValue])

4、findIndex

返回数组中通过测试的第一个元素的索引。

如果找到函数返回 true 值的数组元素，则 findIndex() 返回该数组元素的索引 ，否则返回-1。

注： 不会为没有值的数组元素执行函数。

语法： array.findIndex(function(currentValue, index, arr) [, thisValue])

5、from

从具有 length 属性或可迭代对象的任何对象创建数组 。

语法： Array.from(object [, mapFunction, thisValue])

6、includes

检查数组是否包含指定的元素。

如果数组包含元素，则此方法返回 true，否则返回 false。

注： 方法区分大小写。

语法： array.includes(element [, start])

7、keys

返回 Array Iteration 对象，包含原始数组的键。

语法： array.keys()

8、flat

扁平化数组，返回扁平化后的数组，不改变原数组
接受一个整数作为参数，表示扁平的层数

三、会改变原数组的API

splice、reverse、sort、push、pop、shift、unshift、fill
